# Covid19 Demo App

Covid19 Demo App

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```

### Lint the files
```bash
yarn run lint
```

### Build the app for production
```bash
quasar build
```

### Live demo
[Demo page](https://covid19-demo-app.netlify.app/#/)
